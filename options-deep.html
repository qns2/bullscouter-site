How to read this page

Flow signal: from the latest options flow scan. 🚀 strong bull (>+5) · 🟢 bull (+2 to +5) · ⚖️ neutral · 🔴 bear · 🚨 strong bear.

Skew direction: 25Δ call IV − 25Δ put IV. Bullish = calls bid up (upside speculation). Bearish = puts bid up (hedging / fear).

Skew magnitude: |risk reversal| in vol points. Normal ≤2vp · Moderate 2–4vp · Extreme >4vp.

GEX sign: dealer gamma exposure. Long γ >+$100M (moves dampened). Short γ <−$100M (moves amplified, squeezes likely).

Zero-γ buffer: distance from spot to the dealer-gamma zero crossing. Safe >10% (stable) · Caution 5–10% · Danger <5% (spot can pop regime at any move).

Skew computed only when earnings is within 45 days. All Greeks via Black-Scholes from yfinance IV (or Tradier directly when configured).
